Главное

Вы попали на страницу инструкций по установке и настройке VPN для своих от annndruha. Чтобы быть в курсе обновлений, блокировок и проблем с сервером советую подписаться на мой Telegram-канал. (Только критически важная информация по VPN, без спама).

1. Настойчиво прошу всех пользователей отключать VPN, если в нём отсутствует необходимость. Используйте только для сайтов, которые не открываются без VPN.
2. Постарайтесь не использовать VPN для скачивания торрентов и другой нелегальной деятельности, иначе Нидерланды могут заблокировать сервер.

AmneziaVPN (на протоколе OpenVPN over Cloak) (Пока работает)


AmneziaVPN - OpenSource приложение от российских разработчиков. Оно поддерживает сразу много протоколов на выбор, почти все имеют обфускацию против блокировок.
  • Протокол AmneziaWG - обфусцированный Wireguard (его блочат с мобильных операторов)
  • Протокол XRay - пока работает

Настройка

1. Скачайте и установите на устройство приложение:

  • Клиент для Windows
  • Клиент для macOS
  • Клиент для Linux
  • Клиент для Android
  • Клиент для iOS

2. Получите ключ доступа или файл с ключом доступа annndruha, а затем сохраните его на нужное устройство.

3. Откройте приложение AmneziaVPN

  • Нажмите "+", затем "Добавить новое соединение". Выберите либо файл-ключ.
  • Если Вам нужен VPN только для пары приложений, то ПОЖАЛУЙСТА, включите "Раздельное VPN-туннелирование" и разрешите только определённым приложениям подключаться через VPN. Это снизит нагрузку на сервер и позволит Вам не отключать VPN.


Outline (Частично заблокирован)


Outline - Open-source приложение от дочки гугла основанное на протоколе ShadowSocks.

Протокол устойчив к глубоким блокировкам, но ценой этому является более низкая скорость по сравнению с Wireguard.

Настройка

1. Скачайте и установите на устройство приложение:

  • Outline: Windows Android iOS macOS Linux
  • Shadowsocks: альтернативные клиентские приложения

2. Получите ключ доступа у annndruha, который начинается с ss://, а затем скопируйте его.

3. Откройте клиент Outline или Shadowsocks.

  • Outline: Если ваш ключ доступа определился автоматически, нажмите "Подключиться". Если этого не произошло, вставьте ключ в поле и нажмите "Подключиться".
  • Shadowsocks: Скопируйте ключ в буфер обмена. Найдите кнопку импортировать из буфера обмена. (Import from Clipboard).


Wireguard (Полностью заблокирован)


Wireguard - надёжный простой протокол с высокой пропускной способностью, и низкой задержкой. Однако факт использования Wireguard легко обнаружить и как следствие - заблокировать.

Настройка

1. Скачайте и установите на устройство клиент Wireguard

2. Получите .conf-файл у annndruha.

3. В приложении нажмите профиль/туннель и выберите соответствующий файл.

В случае, если приложение пишет что невозможно импортировать туннель, необходимо переименовать .conf-файл, убрав из названия нижние пробелы.

(Опционально): Кнопку включения/выключения на телефонах можно удобно добавить в шторку для быстрого доступа.

Технические детали

Все VPN сервисы находятся на одном виртуальном сервере (212.80.218.172) в Амстердаме (Нидерланды) и используют общие вычислительные мощности. Сервер арендован у провайдера VDS.SH, доступ к серверу имеет только провайдер и я. Однако ни я, ни провайдер не имеют доступа к передаваемым Вами данным (они зашифрованы), только к количеству используемого трафика. Используйте только https соединение, поскольку http может просмотреть любой узел, через который такой трафик проходит.
