AmneziaVPN


AmneziaVPN - OpenSource приложение от российских разработчиков. Оно поддерживает сразу много протоколов на выбор, почти все имеют обфускацию против блокировок.

  • Протокол AmneziaWG - Не работает на мобильных сетях
  • Протокол XRay - Работает и на домашних и на мобильных сетях

Настройка

1. Скачайте и установите на устройство приложение:

  • Клиент для Windows
  • Клиент для macOS
  • Клиент для Linux
  • Клиент для Android
  • Клиент для iOS

2. Получите ключ доступа или файл с ключом доступа annndruha, а затем сохраните его на нужное устройство.

3. Откройте приложение AmneziaVPN

  • Нажмите "+", затем "Добавить новое соединение". Выберите либо файл-ключ.
  • Если Вам нужен VPN только для пары приложений, то ПОЖАЛУЙСТА, включите "Раздельное VPN-туннелирование" и разрешите только определённым приложениям подключаться через VPN. Это снизит нагрузку на сервер и позволит Вам не отключать VPN.


Технические детали

Все VPN сервисы находятся на одном виртуальном сервере в Амстердаме (Нидерланды) и используют общие вычислительные мощности. Сервер арендован у провайдера VDS.SH, доступ к серверу имеет только провайдер и я. Однако ни я, ни провайдер не имеют доступа к передаваемым Вами данным (они зашифрованы), только к количеству используемого трафика. Используйте только https соединение, поскольку http может просмотреть любой узел, через который такой трафик проходит.
